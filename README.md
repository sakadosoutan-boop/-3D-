# 寝殿造り3D探訪 〜平安貴族の住まいを歩く〜

平安貴族の邸宅様式「**寝殿造り**」を 3D で歩いて学べる、ブラウザ完結型の学習シミュレーターです。
建物・建具・調度品をタップ（クリック）すると解説が読め、名称当てクイズや「貴族の一日」体験、
垣間見ミッション、図鑑などのモードを通して、古文・日本史の知識を体験的に身につけられます。

古文・日本史の教材として、タブレット横向き／PC での利用を想定しています。

## 特長

- **完全オフライン動作**：Three.js（r128）と主要テクスチャを HTML に内蔵。`file://` で直接開けます
  （※環境音 `sounds/` のみ外部ファイル。音を鳴らす場合はフォルダごと同じ場所に置いてください）。
- **外部3Dモデル不使用**：建物・庭園・人物・四神・自然はすべてプログラムによる**手続き的生成**。
- **季節 × 時刻**：春夏秋冬／昼・夕・夜を切り替え、ライティング・霞（フォグ）・環境音が連動。
- **環境音**：季節ごとの鳥・虫の声、管弦（琵琶・箏・笙・龍笛）、水音・焚火などを WebAudio で再生
  （音源が無い環境では合成音にフォールバック）。

## 使い方

1. `寝殿造り3D探訪_統合版.html` をブラウザで開く（ダブルクリックで可）。
2. 音を鳴らす場合は、同じ階層に `sounds/` フォルダを置いておく。
3. タイトルからモードを選んで開始。

### 操作方法

| 環境 | 移動 | 視点 | 調べる |
|------|------|------|--------|
| PC | WASD / 矢印キー（Shiftで早歩き） | 画面ドラッグ | クリック |
| タブレット | 左下のジョイスティック | 画面ドラッグ | タップ／右下「調べる」 |

上部バーから **俯瞰⇄歩く**、**時間**、**季節**、**名称札の表示**、**音 ON/OFF**、**図鑑**、**ヘルプ** を切り替えできます。

## モード

- **自由散策** … 邸内と庭を自由に歩く
- **名称当てクイズ / ⏱ タイムアタック** … 部位・調度品を探してタップ
- **男性貴族の一日 / 女性貴族の一日** … 出仕・教養・管弦などを追体験
- **垣間見ミッション** … 女房と侍従の目を避けて忍ぶ
- **平安ものしり図鑑** … 解説を読んだ項目が記録され、次回起動後も解放状態を保持

## アプリ内部の構成（保守のための地図）

`寝殿造り3D探訪_統合版.html` は単一ファイルですが、概ね次の順で構成されています。

1. **`<head>`** … メタデータ・OGP・favicon
2. **インライン Three.js r128**（`<script>` 冒頭の巨大行）
3. **`<style>`** … タイトル／上部バー／解説ポップアップ／各HUD／図鑑／ジョイスティック等のUI
4. **`<body>`** … タイトル・上部バー・各オーバーレイ（図鑑・解説・クイズ・クエスト・結果・ヘルプ・クレジット）
5. **メインスクリプト**
   - テクスチャ生成（`canv` / `woodTex` ほか）と共通マテリアル `MAT`／`M()`
   - 空・IBL（`makeSkyEnv`）、地形・床高マップ（`addZone` / `groundH` / `inPond`）
   - 造形ヘルパー（`box` / `cyl` / `hipRoof` / `railing` …）と建物群（`building` / `corridor` / 寝殿 `SH`）
   - 人物・霊獣・水鳥・四神・植生・季節演出（`makeHeianFigure` / `applySeason` / `SEASONS`）
   - 音響（`SOUNDSCAPE` / `SFX` / `AmbientAudio`）
   - 状態 `APP`、入力（キー・ジョイ・ドラッグ・ピック `pick`）
   - 図鑑 `CODEX_KEY` / 表示設定の永続化 `PREFS`
   - 各モード（`startQuiz` / `startQuest` / `startKaimami`）
   - メインループ `animate()`、パフォーマンス計測 `__perf`

### 開発メモ

- **パフォーマンス計測オーバーレイ**：`~`（Backquote）キー、または URL に `?fps=1` を付けると
  FPS・draw call・三角形数・ジオメトリ/テクスチャ数を表示。
- **表示設定の永続化**：季節・時刻・名称札はユーザーが上部バーで切り替えたときのみ
  `localStorage`（キー `shinden3d-prefs-v1`）に保存。クイズ等のシナリオ起因の変更では保存しない。
- **アクセシビリティ**：OS の「視差効果を減らす（prefers-reduced-motion）」設定時は UI アニメーションを抑制。
- **耐障害性**：WebGL コンテキスト喪失時は描画を休止し、復帰時に自動再開（タブレットのスリープ対策）。
